# WeNote

## 소개

To-Do List를 간단히 구성한 1인 토이 프로젝트입니다.

개인 노트를 구성할 수 있으며,

프로젝트를 만들어 여러 사람과 같은 목표의 To-Do List를 구성할 수 있습니다.

프론트는 Vercel , 백엔드는 CloudType을 통하여 배포했습니다.
